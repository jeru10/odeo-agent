# 🤖 Agent Odeo — AI Manager pour Restaurants

## Installation en 2 minutes sur n'importe quel PC

```
╔══════════════════════════════════════╗
║    🤖 AGENT ODEO — v1.0             ║
║    Assistant IA pour Restaurants     ║
║    🇲🇦 Maroc — Darija / Français     ║
╚══════════════════════════════════════╝
```

## Fonctionnalités

- 📊 **Analyse des ventes** en temps réel
- 📦 **Stock intelligent** avec alertes automatiques
- 📱 **Rapport WhatsApp** quotidien
- 💬 **Assistant vocal** en darija/français
- 🌐 **Dashboard web** pour le propriétaire
- 📈 **Prédictions** et tendances
- 🧠 **RAG** intégré (menu, tarifs, infos du resto)

## Installation

### Windows
```
install.bat
```

### Linux/Mac
```
bash install.sh
```

## Utilisation

1. Lance l'agent : `python run.py`
2. Ouvre le dashboard : http://localhost:5000
3. Configure l'API Odeo dans settings
4. L'agent travaille 24h/24
